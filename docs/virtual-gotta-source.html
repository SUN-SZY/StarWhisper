Virtual-GOTTA Source Notes

Notes extracted from the presentation PDF 人工智能驱动科教科研, used to update the StarWhisper README and the Virtual-GOTTA interactive map.

Presentation Context

The presentation frames Virtual-GOTTA as an AI-driven virtual Sitian project, dated 2026.5.27, and places it in the broader AI4S discussion around AI and robotic scientists.

PDF pages used: 1-3.

StarWhisper Telescope

  • StarWhisper Telescope is described as an agent-based observation assistant system to approach an AI astrophysicist.
  • The presentation labels the direction as a 具身智能望远镜, meaning an embodied-intelligence telescope workflow rather than only a chat interface.

PDF page used: 5.

Virtual-GOTTA System Role

  • StarWhisper acts as the large-model agent interface for human-machine interaction.
  • The target workflow connects alert release, station information, telescope status, and real-time response.
  • StarWhisper 3.0 first connected large models to astronomical telescopes; StarWhisper 4.0 drove the nearby galaxy survey project and improved the planning-execution-report loop.
  • StarWhisper 5.0+ points toward embodied-intelligence transformation for scientific telescopes.

PDF page used: 6.

Science Application

  • Virtual-GOTTA is described as a multi-agent global transient-source telescope data networking system.
  • The supernova clock is cited as already providing multiple very early supernova candidate alerts within one day of explosion.
